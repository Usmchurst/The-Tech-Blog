# The-Tech-Blog

# Description

A mysql database and CMS-style Blog built using Model View Controller (MVC). Built using MySQL2, Express, Sequelize, Handlebars and dotenv.
This is a CMS-style blog site thas just like a Wordpress site, here developers can publish their blog posts and comment on other developersâ€™ posts also. App is following the MVC in it's architectural structure, it uses Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication. It was developed over some starter code and deployed to Heroku.
